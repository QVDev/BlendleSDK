# BlendleSDK
SDK for Blendle.nl.

##Usage
Gradle

```java
repositories {
    maven { url "https://github.com/QVDev/BlendleSDK/raw/master/Android/SDK/releases/" }
}

dependencies {
    compile 'qvdev.utils.sdk.blendlesdk:blendleandroidsdk:1.0.0-SNAPSHOT'
}
```
